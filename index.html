عن المشروع (About)

مشروع ZeroTrace OS هو مبادرة تقنية وطنية تهدف لتوفير نظام تشغيل آمن ومخصص لاختبار الاختراق والتحليل الجنائي الرقمي. تم العمل عليه بروح الفريق الواحد بين مطورين من سلطنة عُمان وسوريا لتقديم حلول سيبرانية تضاهي الأنظمة العالمية بميزات خصوصية فريدة وتعديلات جوهرية على الأداء.

سجل بناء النظام (Build Code)

UI Customization

تخصيص هوية الـ Bootloader

الأمر المستخدم لتغيير صورة خلفية الـ Grub بنجاح:

sudo cp ~/Downloads/desktop-background.jpg /boot/grub/grub-16x9.jpg && sudo update-grub
System Fix

تنظيف المستودعات وإصلاح الـ APT

حل مشكلة أخطاء 404 التي تظهر عند تحديث النظام:

sudo rm /etc/apt/sources.list.d/cubic.list && sudo apt update
Performance

تسريع الضغط بجميع الأنوية

استخدام المعالج بالكامل لإنتاج ملف الـ ISO بسرعة قصوى:

mksquashfs /source /destination -comp gzip -processors $(nproc)
